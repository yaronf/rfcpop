Version 0.1, 25-Jan-2013

Initial public release.

Scrapes RFCs from tools.ietf.org.

OpenID registration, login, logout.

Comments can be attached to each section heading.

Wysiwyg editor for comments.

Comments sanitized before they are displayed, to avoid XSS attacks.

URLs sanitized. For now, only IETF URLs are allowed.

Anybody can view comments, but you need to login to add new ones.

Comments can only be edited/deleted by their author.
